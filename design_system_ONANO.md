# ONANO --- Design System Oficial

Versión 2.0 --- Mobile First

Este documento define las reglas visuales, técnicas y estructurales que
deben seguir diseñadores UI/UX y desarrolladores.\
El objetivo es garantizar coherencia, elegancia y escalabilidad en todos
los productos digitales de ONANO.

------------------------------------------------------------------------

# Principio Fundamental

## Mobile First

Todos los diseños deben comenzar desde resolución móvil y escalar
progresivamente hacia tablet y desktop.

Reglas clave: - Priorizar claridad y jerarquía en pantallas pequeñas. -
Evitar sobrecarga visual. - Usar espaciado generoso. - Componentes
pensados para interacción táctil. - Escalado progresivo mediante
breakpoints consistentes.

# Psicología del Color

La identidad cromática de ONANO se basa en tonos azules profundos
combinados con acentos luminosos.

El azul transmite:

-   Ciencia
-   Precisión
-   Estabilidad
-   Credibilidad
-   Evolución tecnológica

Los contrastes refuerzan claridad estructural, jerarquía visual y
sensación de control.

# Sistema de Bordes y Radios

## Regla General

Todo elemento con altura mayor a 48px debe tener:

border-radius: 24px;

Esta regla aplica a: - Cards - Inputs grandes - Contenedores
interactivos - Componentes destacados

------------------------------------------------------------------------

# Sistema de Botones

## Botón Estándar

-   Altura máxima: 48px
-   Border radius: 24px
-   Uso: acciones secundarias o funcionales
-   Debe mantener proporciones táctiles adecuadas para mobile

------------------------------------------------------------------------

## Botón C-T-A (Call To Action)

-   Altura máxima: 64px
-   Border radius: 32px
-   Uso: acción principal de pantalla
-   Debe destacarse claramente en jerarquía visual

------------------------------------------------------------------------

# Lineamientos Generales de UI

-   Estilo minimalista y premium.
-   Espaciado generoso.
-   Jerarquía clara.
-   Uso intencional del color.
-   Componentes táctiles cómodos.
-   Consistencia visual en todos los puntos de contacto.
-   Evitar saturación visual.
-   Priorizar claridad sobre ornamentación.

------------------------------------------------------------------------

Este Design System es un documento vivo y debe evolucionar junto con el
crecimiento tecnológico y estratégico de ONANO.
